# План разработки: CS:GO Cases Simulator

## 🎯 Описание проекта

Симулятор открытия кейсов в стиле CS:GO с элементами казино-тематики. Фронтенд-приложение на React с использованием библиотеки react-bits для красивых анимаций. Дизайн: футуристический, тёмный, сочный — в духе Apple/Google нового поколения.

---

## 🏗️ Архитектура MVP

### Стек технологий

- **React 18+** — основной фреймворк
- **TypeScript** — типизация
- **Vite** — сборщик
- **Tailwind CSS** — стилизация
- **react-bits** — анимации и визуальные эффекты (через shadcn registry)
- **Zustand** — управление состоянием
- **React Router v6** — роутинг между страницами
- **localStorage** — сохранение прогресса (инвентарь, баланс)

## 📦 Функционал MVP

### 1. Главная страница (Home)

- Отображение текущего баланса (**стартовый: $10,000**)
- **Кнопка "Пополнить" — добавляет +$1,000 к балансу** (можно нажимать многократно)
- Лента последних выпавших предметов (live feed, последние 10 открытий)
- CountUp анимация при изменении баланса
- Навигация по разделам

### 2. Открытие кейсов (Cases)

- Каталог кейсов с разной ценой и редкостью
- Анимация открытия (горизонтальная рулетка, скролл предметов)
- Шансы выпадения по редкостям:
  - Common (серое) — 50%
  - Uncommon (синее) — 25%
  - Rare (фиолетовое) — 15%
  - Mythical (розовое) — 7%
  - Legendary (красное) — 2.5%
  - Ancient (золотое) — 0.5%
- Полученный предмет попадает в инвентарь
- **Смесь реальных CS:GO скинов (AK-47 Redline, AWP Dragon Lore и т.д.) и оригинальных придуманных**

### 3. Инвентарь (Inventory)

- Отображение всех полученных предметов с 3D-карточками (TiltedCard)
- Возможность продать предмет (50% от стоимости)
- Фильтрация по редкости
- Статистика: всего предметов, общая стоимость

### 4. Краш (Crash)

- График с растущим множителем (x1.0 → x∞)
- **Ставка игрока + симуляция 3–5 ботов (с никами, аватарами, авто-кэшаут)**
- Нужно успеть забрать выигрыш до "краша"
- Краш происходит в случайный момент (алгоритм с house edge ~5%)
- **Live-чат ботов с реакциями** (проиграл/выиграл)

### 5. Рулетка (Roulette)

- Ставка на цвет (красное/чёрное/зелёное)
- Анимация вращения колеса (горизонтальная лента)
- Выплаты: **2x для цветов, 14x для зелёного**
- **Валюта: доллары ($)**

## 🎨 Дизайн-система

### Цветовая палитра

- **Фон**: `#0a0a0f` (почти чёрный с синим)
- **Поверхности**: glassmorphism с `backdrop-blur`
- **Акцент 1**: `#6366f1` (индиго — primary)
- **Акцент 2**: `#8b5cf6` (фиолетовый)
- **Акцент 3**: `#06b6d4` (cyan — highlight)
- **Успех**: `#10b981` (зелёный)
- **Опасность**: `#ef4444` (красный)
- **Золото**: `#f59e0b` (ancient/legendary)

### Редкости (цвета предметов)

| Редкость  | Цвет              | Hex       |
|-----------|-------------------|-----------|
| Common    | Серый             | `#b0b0b0` |
| Uncommon  | Синий             | `#4d9fff` |
| Rare      | Фиолетовый        | `#8847ff` |
| Mythical  | Розовый           | `#d32ee6` |
| Legendary | Красный           | `#eb4b4b` |
| Ancient   | Золотой/Оранжевый | `#e4ae39` |

### React-bits компоненты

- `Aurora` — фоновая аврора (главный фон приложения)
- `SpotlightCard` — карточки кейсов и инвентаря
- `GradientText` — градиентные заголовки
- `ShinyText` — блестящий текст для балансов
- `DecryptedText` — hacker-style текст при загрузке/навигации
- `ClickSpark` — искры при нажатии кнопок
- `AnimatedContent` — анимации появления элементов
- `GlassSurface` — glassmorphism панели
- `CountUp` — анимация счётчика баланса
- `ElectricBorder` — рамка для ancient/legendary предметов
- `SplashCursor` — кастомный курсор с жидким эффектом
- `Noise` — зернистый оверлей для глубины
