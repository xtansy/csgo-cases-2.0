# Архитектура

## Управление состоянием

Единый Zustand-стор (`src/store/gameStore.ts`), сохраняемый в localStorage:

```
balance: number           — текущий баланс игрока
inventory: InventoryItem[]  — все полученные предметы (с instanceId)
feed: FeedEntry[]          — последние 20 выпавших предметов (живая лента)
```

Ключевые действия: `topUp()`, `deduct(amount)`, `addItem(item)`, `sellItem(instanceId)`, `pushFeed(entry)`

## Рандомизация

`src/lib/randomize.ts`:

- `rollRarity()` — взвешенный случайный выбор из RARITY_WEIGHTS
- `rollItem(items)` — выбирает случайный предмет выпавшей редкости; при пустом пуле переходит к более низкой редкости
- `generateRouletteItems(caseItems, winner)` — формирует ленту рулетки из 80 предметов с победителем на индексе 79
- `generateCrashPoint()` — экспоненциальное распределение с 5% шансом мгновенного краша (преимущество казино)

## Маршрутизация

Вложенные маршруты React Router v6 внутри `<Layout>`:

```
/           → Home
/cases      → Cases catalog
/cases/:id  → CaseOpen (анимация рулетки)
/inventory  → Inventory
/crash      → Crash game
/roulette   → Roulette game
```

## Ключевые компоненты

### Layout

Глобальная обёртка, содержащая:

- Aurora (WebGL-фон, фиксированный, z-0)
- Noise (плёночное зерно, фиксированный, z-1)
- SplashCursor (эффект курсора, глобальный)
- ClickSpark (обёртывает весь основной контент)
- Navbar (фиксированный снизу)

### CaseRoulette

Анимация на чистом CSS/JS с использованием `translateX` с cubic-bezier `(0.12, 0.8, 0.25, 1)` в течение 5 секунд. Победитель размещается на индексе 79, смещение цели рассчитывается для центрирования во viewport.

### Crash

Использует `setInterval(50мс)` для инкремента множителя по формуле `e^(0.15 * elapsed)`. Canvas отрисовывается каждый тик с градиентной заливкой и обводкой. Боты проверяют свои пороги cashout на каждом тике.

### Roulette

Лента из 120 предварительно сгенерированных тайлов. При вращении применяется большое `translateX` для имитации замедления, останавливаясь на заранее выбранном целевом индексе тайла.
